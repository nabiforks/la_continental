Car Repair Management

This app is fantastic solution for repair industry , which include any sort of machine repair , Here car repair specifically focus on automobile and fleet repair, which cover all aspect of repairing process it is good solution for repair industry which include , car workshop, automobile spartshop, garage, for any car that needs repairing , generated diagnosis and work order , system will create Sale order and Invoice for the whole process done on the Car.

Features

Create car repair order

You can create car repair order with car related information.

Add car service details.

You can add car service details like (Car guarantee,Guarantee type).

Create car diagnosis.

You can create diagnosis based on car repair order.

Manage car workorder.

You can manage car repair work order (start,pending,resume,cancel).

Print Report.

You can print car repair,dignosis, workorder reports.

Create Car Diagnosis.

You can create car diagnosis based on car repair order.

Assign to Technician

You can assign car diagnosis to technician.

Fully Automated workflow.

Repair order, Diagnosis and workorder are fully automated with workflow process, So when anything get effected on any document should be reflected on repair process. I.e When Quotation created and sent, it change stage as "Quotation Sent" on repair, when workorder finsihed it change as Workorder completed.

Access Rights and Roles

We have four different access groups for machine repair module.


  • Director Commercial
  • Service Manager
  • Head Technician
  • Technician

Roles


  • Technician : Technician can work on work order.He/She can only see his/her work order and only work on his/her assigned work order.Technician can only have reading access to diagnosis and repair order.

  • Head Technician : Head technician can see all work order also able to assign a technician on a dignosis and work order. He/she has access to enter dignosis result. He/She cannot change any thing on a repair order.

  • Service Manager : Service Manager creates repair order.convert it as a dignosis able to create quotations.

  • Director Commercial :Director Commercial is the admin of this system he/she has full access rights of every thing.
